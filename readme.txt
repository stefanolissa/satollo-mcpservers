Enable the creation of MCP servers to expose abiltities to AI agents

== Description ==

Using the WP MCP adapter library, it lets the creation of MCP servers to expose selected abilities.

MCP servers are the way AI agents can connect to a site for data or action execution (of course only with authentication).

= Warning =

Abilities are registered an implemented by plugins and themes: check carefully what they do before exposing!
